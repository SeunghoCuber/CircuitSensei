The Problem
The Learning Gap in Electronics
01
No Spatial Awareness
Beginners stare at a breadboard and don't know where to place components. Diagrams don't map to physical rows and columns.
02
No Real-Time Feedback
Traditional tutorials are static. Students make wiring mistakes and have no way to verify correctness before applying power.
03
Dangerous Errors
A single miswired LED or short circuit can destroy components or cause smoke. Fear of breaking things stops learners cold.
The Solution
Meet CircuitSensei
CircuitSensei takes your circuit goal and component inventory in plain English — then acts as a real-time AI mentor, drawing guidance directly onto a live webcam view of your breadboard.

No prior electronics knowledge required.

Natural Language Planning

Describe your goal. CircuitSensei derives a safe, step-by-step placement plan with exact row/column coordinates.

On-Screen Visual Annotation

Highlighted holes, arrows, and labels are drawn over the live camera feed — no LED strips, no special hardware.

Gemini Vision Verification

Before each step advances, Gemini Vision inspects the board image to confirm correct placement.

Architecture
The Agentic Workflow
STATE 1
Intake
User states a circuit goal and lists their components in plain English. Agent parses both.
›
STATE 2
Plan
Agent derives a breadboard placement plan. Hole conflicts are auto-repaired using topology rules.
›
STATE 3
Instruct
One physical action at a time. Annotated overlay highlights exactly which holes to use.
›
STATE 4
Verify
Gemini Vision inspects the webcam frame. Step only advances after visual confirmation passes.
›
STATE 5
Test
Arduino connects via USB serial. Agent runs voltage, LED, and button tests on the finished circuit.
Feature Deep-Dive
Visual Guidance
in Action

Agent maps breadboard coordinates to pixel positions in the webcam frame using a calibrated grid.

Arrows, labels, and highlighted holes are drawn directly on the captured image — no special hardware needed.

Safety net: if the image is too dark, auto-brightness enhancement kicks in before Gemini sees the frame.

Emergency keyword detection — mentions of smoke, heat, or burning immediately trigger a disconnect warning.

Simulated annotated guidance overlay — webcam view with highlighted placement holes, component arrow, and step message
Under the Hood
Technology Stack
AI
Gemini 2.0
Agent brain + Vision verification. Parses goals, plans circuits, and inspects webcam frames for correct placement.
PY
Python 3.11
Core agent loop, state machine, tool execution, and session management.
CV
OpenCV
Webcam capture, frame enhancement, and annotation drawing — holes, arrows, and labels overlaid in real time.
⚡
Arduino
USB serial testing after visual verification. Runs voltage divider, LED drive, and button read tests.
⚛
React + Vite
Live Camera Feed UI for the frontend — real-time annotated frame display and session control.
∞
State Machine
7-state finite machine (IDLE → TEST) with validated transitions. Prevents unsafe Arduino activation before visual sign-off.
🔓
Open Source
MIT licensed. Mock mode runs without a Gemini key or hardware — zero-friction for first-time contributors.
🔓
pytest
State machine, safety response, annotation drawing, coordinate mapping, and mock Arduino paths are all test-covered.
